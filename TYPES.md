## 类型
### 简单类型
这些类型被存储为值。
#### 空类型
没有类型名称，  
字面量为`null`。
#### 布尔类型
类型名称为`boolean`，  
字面量为`false`或`true`。
#### 数值类型
##### 字面量
- `-?[0-9]+(\.[0-9]+)?` 
  - 十进制字面量
- `-?0[bB][0-9]+`
  - 二进制字面量
- `-?0[xX][0-9]+`
  - 十六进制字面量

##### 整数
不支持带小数点的字面量。
|类型|说明|
|--|--|
|i8|有符号8位整数|
|u8|无符号8位整数|
|i16|有符号16位整数|
|u16|无符号16位整数|
|i32|有符号32位整数|
|u32|无符号32位整数|
|i64|有符号64位整数|
|u64|无符号64位整数|

##### 小数
|类型|说明|
|--|--|
|f32|32位小数|
|f64|64位小数|
#### 字符串
类型名称为`string`。内部存储为`u8`数组。  
字面量为`"`，`'`或`` ` ``包裹的字符序列。
##### 转义字符
- `\\`
  - 用于表示字面含义的`\`
- `\u{<hex>}` or `\U{<hex>}`
  - 用于直接指定unicode码点
- `\x<hex>` or `\X<hex>`
  - 直接指定ASCII码点
  - `<hex>`: 二位十六进制
- `\'`, `\'`, `` \` ``
- `\r`, `\t`, `\n`
##### 格式化
以`` ` ``包裹内容时，支持在字符串中包含：
`${<expression>}`。  
其中的表达式必须为字符串，或者实现了`StringConvertible`接口。
```
interface StringConvertible {
    fun to_string(): string;
}
```
### 数组
- `<type>[]`
  - 变长数组
- `<type>[<length>]`
  - 定长数组
